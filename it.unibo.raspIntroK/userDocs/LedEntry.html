The Led cathode is the shortest terminal (on the flat site) and must be connected to ground. The other terminal is the anode and must connected to the voltage source (e.g. pin 25 or pin 18 PWM).

A Led has a operating voltage (2.2V) and a current rating (18 mA). We could need a resistor to bring down the voltage to the current level. For example, if we connect the anode to 5V, then we need a 155 ohm ( (5-2.2)/0.018 ) resistor.

Modeling a led (domain model)

  • For a software designer, a physical led (as any other any output device) is always associated with some basic software (usually a driver).
  • A driver for an output device usually provides methods that work as state modifiers.

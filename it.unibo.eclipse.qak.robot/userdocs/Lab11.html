Goal

Design and build a software support for the development of (IOT) applications involving one or more Differential Drive Robots (DDR). The support must help an application designer in the following tasks:
  1. sendMoveCmds: send commands to move a DDR
  2. receiveSensorInfo: receive (and show) information from the sensors put on the DDR.
  3. handleEnvCond: allow a robot to be sensible to environment conditions, e.g. the value of the ambient temperature, an alarm emitted by some external entity, etc.
  4. manyDDRTypes: allow the usage of bot virtual and real robots built with different technologies

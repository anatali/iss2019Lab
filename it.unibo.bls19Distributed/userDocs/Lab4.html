72939 - LAB4 | A ButtonLedSystem distributed

LabSS-lectures site

REQUIREMENTS

Design and build a distributed ButtonLed software system (BLSD) in which the Led and the Button are virtual or real devices, each connected to a different computational device. The application logic should run on a third computational device, e.g. a conventional PC.

START UP

  1. In it.unibo.bls19Distributed, execute gradle init.
  2. Open IntelliJ IDEA on this new project:
  3. Update gitignore.
  4. Create the directory src/main/java the directory libs and (optionally) userDocs.
  5. Copy the library it.unibo.bls19Local-1.0.jar built during our previous work (Lab3.html) into the local directory libs.
  6. Define proper dependences in build.gradle, in particular the 'CUSTOM' section.
  7. Define a simple application (e.g. StartTest) to check that all is ok.
